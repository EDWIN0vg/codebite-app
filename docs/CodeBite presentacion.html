CODEBITE v1.0
EL PROBLEMA

El tiempo muerto es
tiempo desperdiciado.

Los desarrolladores en etapa de formación y los profesionales junior pierden horas diarias en transporte y esperas. Sin herramientas móviles técnicamente rigurosas, ese tiempo es irrecuperable.

72min
Tiempo promedio diario invertido en transporte urbano por trabajadores de tecnología en Colombia. Tiempo productivo no aprovechado.
58%
Tasa de olvido técnico pasado el 70% del período de inactividad cognitiva, según la curva de Ebbinghaus aplicada al aprendizaje de código.
3x
Mayor retención de conceptos técnicos cuando el aprendizaje ocurre en sesiones distribuidas cortas versus una sola sesión prolongada.
0
Apps nativas en Android que combinen micro-retos de código real con validación técnica, gamificación y feedback educativo personalizado por nivel.

// Brechas Identificadas en el Ecosistema Actual

  • Las plataformas existentes (HackerRank, LeetCode) no están diseñadas para sesiones de 2 minutos en movilidad.
  • Ausencia de sintaxis coloreada real en apps educativas móviles: el texto plano destruye la experiencia de código.
  • No existe diferenciación de niveles (Junior / Mid / Senior) dentro del flujo de un único reto.
  • La gamificación superficial premia la participación, no la corrección. El XP no distingue entre respuesta acertada y fallida.
  • Cero feedback educativo post-quiz: el usuario sabe si falló, pero no por qué falló ni cómo corregirlo.
  • Las soluciones existentes son web-first: sin soporte offline, tiempos de carga lentos y UX no optimizada para pulgar.
LA SOLUCIÓN

Retos de código real.
En 2 minutos.
En tu bolsillo.

CodeBite transforma los tiempos muertos en sesiones de práctica técnica intensiva: micro-retos curados, editor visual estilo IDE, sistema de XP honesto y retroalimentación educativa inmediata.

IMPLEMENTADO

Autenticación y Gestión de Sesión

Pantalla de Login nativa con validación de credenciales y persistencia de sesión. El estado del usuario se mantiene entre aperturas de la aplicación, gartantizando una experiencia fluida y continua.

IMPLEMENTADO

Especialización Dinámica por Nivel

Selector interactivo que ajusta el contenido según el perfil del desarrollador: Junior → Mid → Senior. El mismo lenguaje presenta retos de complejidad creciente, permitiendo un camino de aprendizaje progresivo.

IMPLEMENTADO

Visor de Código Estilo IDE

Componente custom que renderiza sintaxis real con coloreado semántico (tema Darcula). Soporta Java, Python y C++. Elimina el texto plano y emula la experiencia visual de un editor profesional en mobile.

IMPLEMENTADO

Sistema de XP Honesto

La lógica de gamificación está corregida: +50 XP únicamente cuando la respuesta es correcta. El sistema no premia la participación ciega — premia el dominio técnico verificado.

Feedback Educativo Post-Quiz: Tras cada respuesta, CodeBite despliega una explicación técnica contextualizada — no solo "correcto/incorrecto". El desarrollador comprende por qué la lógica funciona o por qué su razonamiento fue erróneo, reforzando la retención de conocimiento de forma activa.

ARQUITECTURA Y TECNOLOGÍAS

Stack moderno.
Calidad verificable.

Cada decisión tecnológica responde a un criterio de producción profesional. No se usó ninguna herramienta por conveniencia: cada elección tiene justificación arquitectural.

// Stack Tecnológico
Kt

Kotlin

Lenguaje oficial Android. Null-safety, coroutines, expresividad funcional. Reduce bugs en tiempo de compilación vs Java.

JC

Jetpack Compose — Material 3

UI declarativa reactiva. Elimina XML. Recomposición inteligente: la UI se actualiza solo donde el estado cambia.

JU

JUnit 4 — Pruebas Unitarias

Suite de tests que valida la lógica de negocio crítica: cálculo de XP, validación de respuestas, transiciones de estado.

GitHub Actions — CI/CD

Pipeline automatizado que ejecuta tests en cada push. Cada commit recibe ✅ o ❌. La rama main siempre está en estado válido.

SLIDE 05 / GUÍA LIVE DEMO

Estructura del
Demo en Vivo.

Recorrido de presentación de 5 pasos diseñado para mostrar cada característica crítica en el orden de mayor impacto narrativo. Tiempo estimado: 6–8 minutos.

PASO 01
Pantalla de Login y Sesión
  • Abrir app en frío
  • Ingresar credenciales
  • Mostrar pantalla principal
  • Resaltar persistencia
PASO 02
Selector de Nivel Dinámico
  • Abrir selector de nivel
  • Elegir "Junior"
  • Mostrar reto resultante
  • Cambiar a "Senior"
PASO 03
Visor IDE — Darcula
  • Mostrar fragmento Python
  • Señalar coloreado sintáctico
  • Comparar vs texto plano
  • Scroll por el código
PASO 04
Quiz + Feedback Educativo
  • Responder incorrectamente
  • Mostrar feedback de error
  • Responder correctamente
  • Mostrar feedback de acierto
PASO 05
XP Ganado y Resumen
  • Mostrar pantalla de XP
  • Confirmar: +50 solo si correcto
  • Ver historial de racha
  • Cerrar demo

// Tip Crítico para la Presentación

Antes del demo: ejecuta la app al menos una vez para tener datos cacheados. Durante el paso 4: responde primero incorrectamente — el contraste visual del feedback negativo vs positivo es lo que más impacto genera. Cierra fuerte: vuelve a GitHub Actions y muestra el pipeline verde en vivo desde el navegador del dispositivo o proyector.
