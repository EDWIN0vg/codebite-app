# Ideas de proyecto

1. AlgoBit (Entrenamiento de Algoritmos):

* **Enfoque**: Preparación para entrevistas técnicas en "tiempos muertos".
* **Mecánica**: Desafíos de lógica y estructuras de datos. Incluye una función llamada "Trace the Code", donde el usuario debe predecir la salida de un bucle o recursión antes de que el tiempo se agote.
* **Valor**: Entrena la velocidad mental y la lectura de código complejo.

2. RefactorSnap (Clean Code & Revisión):

* **Enfoque**: Mejora de la calidad de código y buenas prácticas (SOLID, Clean Code).
* **Mecánica**: Se muestra un fragmento de código "sucio". El usuario debe deslizar (estilo Tinder) para identificar si el código es correcto o si contiene un code smell, seleccionando luego la refactorización adecuada.
* **Valor**: Ideal para desarrolladores que buscan subir de nivel de Junior a Mid/Senior.

3. ComposeLab (UI & UX Challenge):

* **Enfoque**: Dominio de interfaces declarativas con Jetpack Compose.
* **Mecánica**: Retos visuales donde se muestra un diseño y el usuario debe elegir los modificadores o parámetros correctos (Arrangement, Alignment, Clip) para replicar la UI propuesta.
* **Valor**: Refuerza la comprensión del sistema de diseño y la jerarquía de vistas de forma visual.
